## 🎯 人脸识别应用方案

### 方案1: 通用目标检测中的人脸检测
```python
from ultralytics import YOLO

# 加载预训练模型
model = YOLO("yolo11n.pt")

# 检测图片中的所有目标（包括人脸）
results = model("your_image.jpg")

# 筛选出人脸（person类别）
for result in results:
    for box in result.boxes:
        if model.names[int(box.cls)] == "person":
            print(f"检测到人脸，置信度: {box.conf}")
```
